04 : Result: Autoencoder

Methodology

Autoencoder is a neural network in which the input is copied to the output. The main functions of the autoencoder are forward propogation and back propogation.

The network works by forward propogating to the output and then taking the difference between the expected output(i.e. the input) and the actual output as the input for back propogation of network.

This process is repeated for the specified epochs till we reach final step where we don't back propgate after the forward propogation as the values of output are close enough to the input.

Within this network, the propogation takes place through a hidden layer with a certain number of neurons in it. On completion of the autoencoder network, these hidden layer values are extracted by us and are taken as new features

These new features are used by us as dataset for classification.



New features

The new features are extracted from the autoencoder neural network and kept in a csv file adding labels f1-f6 to them. These features are later used for classification.

We also have alook at various Scatter plots made from these features

Conclusion

As we can observe from accuracy values and scatter plots the galaxy objects are better separated and hence have better accuracy, and star and quasar accuracy and decluttering has taken place not to the desired level but to a satisfying level leading average acuuracies in 90's for both.
