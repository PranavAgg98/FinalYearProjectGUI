03 : Result: ZCA

Methodology

The first step was performing mean normalization, using which a covariance matrix was computed, and Singular Value Decomposition (SVD) was applied on the covariance matrix to bring out eigenvalues and eigenvectors of the data.

Then, to achieve decorrelation between features, the dataset was projected onto the eigenvectors obtained in the previous step. In the next step, as a part of regularization process, the data were divided by the square-root of eigenvalues (after adding a small constant epsilon, âˆˆ ).

The decorrelated data is put into a Neural network classifier. The Classifier used is a 3 hidden layer neural network with 25, 10 and 8 neurons respectively. It is run for 100 epochs.

Pre-ZCA Classification

The dataset is put into the neural network classifier without applying ZCA for decorrelation. We can observe the plot of the data before ZCA.

As we can see from table below the average accuracy of classification is observed to be 92.1%

Conclusion

ZCA hasn't helped with the decluttering of the data as the data seems to have adjusted in ratio with the original data leading to an extremely similar 3-D plot for same features.

The accuracy has remain largely unchanged with it taking a slight dip.

ZCA hasn't helped as we hoped and hence,we will attempt Autoencoders in attempt for better results
